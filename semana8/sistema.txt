Comando: uname -a
Muestra información del sistema operativo, como el nombre del kernel, la versión, la arquitectura y otros datos del sistema.

Comando: whoami
Muestra el nombre del usuario que está utilizando la sesión actual.
Es útil conocer esta información porque permite identificar el sistema y el usuario que está administrando el servidor, facilitando la configuración, el mantenimiento y la solución de problemas.
